## Comment obtenir les images

### Option 1 : Télécharger depuis le site actuel
1. Allez sur troubles-succion-oralite.ch
2. Clic droit sur chaque image → "Enregistrer l'image sous"
3. Placez-les dans les dossiers correspondants

### Option 2 : Utiliser des images libres de droits
- Unsplash.com
- Pexels.com
- Pixabay.com

Recherchez : "baby eating", "baby tongue", "breastfeeding", "medical professional", etc.

## Dimensions recommandées
- **Hero** : 1920 x 400px (format paysage large)
- **Content** : 800 x 600px (format horizontal)
- **Team** : 600 x 600px (format carré)
- **Logo** : 200 x 50px (fond transparent PNG)

## Format
- JPG pour les photos
- PNG pour le logo (avec transparence)
- WebP pour optimisation (optionnel)
