## 🚀 DÉPLOIEMENT

### Étape 1 : Upload
1. Extraire nouveau-site.zip
2. Upload sur Hostinger via FTP ou File Manager
3. Placer dans `/public_html/`

### Étape 2 : Base de données
1. Créer une base MySQL dans hPanel
2. Importer `api/database.sql`
3. Éditer `api/config.php` avec vos identifiants

### Étape 3 : Stripe
1. Ajouter vos clés Stripe dans `api/config.php`
2. Créer le produit "30 CHF/an" dans Stripe
3. Copier le Price ID
